Bài tập thực hành HTML, CSS và JavaScript cơ bản

Thời hạn: 3 ngày (khoảng 6-8 giờ học và thực hành)

Mục tiêu:

  • Hiểu và sử dụng HTML để tạo cấu trúc trang web.
  • Áp dụng CSS để định dạng giao diện.
  • Sử dụng JavaScript cơ bản để thêm tính năng tương tác.
  • Kết hợp Git/GitHub để quản lý và chia sẻ dự án.

Yêu cầu chi tiết

1. Tìm hiểu về HTML, CSS và JavaScript cơ bản

  • Tìm hiểu các khái niệm cơ bản và ghi chú ngắn (2-3 câu) về:
    • HTML: Cấu trúc trang web, các thẻ cơ bản như <div>, <h1>, <p>, <button>.
    • CSS: Định dạng giao diện, các thuộc tính như color, background, margin, padding.
    • JavaScript: Tương tác, các khái niệm như biến, hàm, sự kiện (onclick).
  • Xem ít nhất 1 video hướng dẫn (gợi ý: “HTML & CSS Tutorial” hoặc “JavaScript Basics” trên YouTube từ kênh freeCodeCamp hoặc F8 Official).
  • Ghi lại 1 điều thú vị học được từ video hoặc tài liệu (gửi qua file text hoặc ảnh chụp).

Tài liệu tham khảo:

  • Website: W3Schools (HTML, CSS, JavaScript tutorials).
  • Bài viết: Tìm “HTML CSS cơ bản” hoặc “JavaScript cơ bản” trên Viblo.
  • Video: Kênh freeCodeCamp hoặc F8 Official (nếu cần phụ đề tiếng Việt).

2. Chuẩn bị môi trường làm việc

  • Cài đặt trình soạn thảo code (gợi ý: Visual Studio Code, tải từ code.visualstudio.com).
  • Đảm bảo đã cài Git và có tài khoản GitHub (từ bài tập trước).
  • Kiểm tra cấu hình Git: 
    git --version
git config --list

3. Tạo dự án trên GitHub

  • Tạo một repository trên GitHub với các yêu cầu:
    • Tên: learn-web- (ví dụ: learn-web-hoang).
    • Chọn public hoặc private tùy ý.
    • Tạo file README.md với nội dung:
      • Tiêu đề dự án (ví dụ: “Dự án thực hành HTML, CSS, JavaScript của Hoang”).
      • Mô tả ngắn (2-3 câu về mục đích dự án).
      • Thông tin cá nhân (tên, ngày hoàn thành).
  • Thêm email pavennt@gmail.com làm collaborator:
    • Vào repository > Settings > Collaborators > Add people > Nhập email.

4. Xây dựng trang web đơn giản

  • Tạo một dự án web với các file:
    • index.html: Cấu trúc trang web.
    • styles.css: Định dạng giao diện.
    • script.js: Thêm tính năng tương tác.
  • Yêu cầu cụ thể:
    • HTML: Tạo một trang web có ít nhất:
      • Tiêu đề (<h1>).
      • Đoạn văn giới thiệu bản thân (<p>).
      • Một nút bấm (<button>) để kích hoạt tính năng JavaScript.
    • CSS: Định dạng trang web:
      • Đổi màu nền hoặc màu chữ.
      • Căn giữa tiêu đề và đoạn văn.
      • Thêm hiệu ứng khi di chuột vào nút (hover, ví dụ: đổi màu nền).
    • JavaScript: Thêm tính năng tương tác:
      • Khi bấm nút, hiển thị thông báo (alert) hoặc thay đổi nội dung một phần tử (ví dụ: đổi nội dung đoạn văn).
  • Tham khảo ví dụ: 
    <!-- index.html -->
<!DOCTYPE html>
<html lang="vi">
<head>
    <meta charset="UTF-8">
    <title>Trang Web Đầu Tiên</title>
    <link rel="stylesheet" href="styles.css">
</head>
<body>
    <h1>Chào mừng đến với trang web của tôi</h1>
    <p id="intro">Tôi là [Tên cháu], đang học CNTT năm 3.</p>
    <button onclick="changeText()">Nhấn để thay đổi</button>
    <script src="script.js"></script>
</body>
</html>

<!-- styles.css -->
body {
    font-family: Arial, sans-serif;
    text-align: center;
    background-color: #f0f0f0;
}
h1 {
    color: #333;
}
button {
    padding: 10px 20px;
    background-color: #2196F3;
    color: white;
    border: none;
    cursor: pointer;
}
button:hover {
    background-color: #0b7dda;
}

<!-- script.js -->
function changeText() {
    document.getElementById("intro").innerText = "Cảm ơn bạn đã nhấn nút!";
}

5. Đẩy code lên GitHub

  • Thực hiện ít nhất 3 commit với mô tả rõ ràng:
    • Commit 1: “Thêm file index.html cơ bản”.
    • Commit 2: “Thêm file styles.css và định dạng giao diện”.
    • Commit 3: “Thêm file script.js và tính năng tương tác”.
  • Đẩy code lên repository:
    • Khởi tạo repository cục bộ: 
      git init
    • Thêm file và commit: 
      git add .
git commit -m "Mô tả thay đổi"
    • Kết nối và đẩy lên GitHub: 
      git remote add origin https://github.com/ten-chau/learn-web-.git
git push -u origin main
  • Nếu gặp lỗi xác thực, tạo Personal Access Token (trong Settings > Developer settings trên GitHub).

6. Nộp bài

  • Gửi link repository GitHub (ví dụ: https://github.com/ten-chau/learn-web-) qua email hoặc tin nhắn.
  • Gửi file ghi chú về HTML, CSS, JavaScript và 1 điều thú vị học được (text, ảnh chụp, hoặc file PDF).
  • (Tùy chọn) Triển khai trang web lên GitHub Pages:
    • Đẩy code vào nhánh gh-pages hoặc bật GitHub Pages trong Settings.
    • Gửi link trang web (ví dụ: https://ten-chau.github.io/learn-web-).

Lưu ý quan trọng

Thời gian: Hoàn thành trong 3 ngày (khoảng 6-8 giờ học và thực hành).

Xử lý lỗi: Nếu gặp lỗi (HTML không hiển thị, CSS không áp dụng, JavaScript không chạy), kiểm tra trên trình duyệt (F12 > Console) hoặc hỏi trên nhóm cộng đồng CNTT (Facebook, X).

Công cụ hỗ trợ:

  • Visual Studio Code với extension Live Server để xem trước trang web.
  • GitHub Desktop để quản lý Git dễ hơn.

Kiểm tra kết quả: Repository sẽ được kiểm tra qua GitHub (commit, file, collaborator) cùng file ghi chú. Trang web sẽ được mở để kiểm tra giao diện và tính năng.
